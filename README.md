# 📊 Customer Behavior Data Analysis Project

##  Overview

This project focuses on analyzing customer purchasing behavior using a transactional dataset. The objective is to extract meaningful insights related to customer segments, product performance, and revenue trends, and present them through interactive dashboards and reports.

The project follows an end-to-end data analysis workflow including data preprocessing, exploratory data analysis (EDA), SQL-based querying, and data visualization.

---

##  Dataset

The dataset contains customer transaction records with features such as:

* Customer demographics (age, gender, location)
* Purchase details (product, category, amount)
* Behavioral metrics (purchase frequency, ratings, discounts, shipping type)

It provides a comprehensive view of customer shopping patterns and business performance.

---

##  Tools & Technologies

* **Python** – Data loading, cleaning, and EDA (Pandas, NumPy, Matplotlib/Seaborn)
* **PostgreSQL** – Writing and executing SQL queries for business insights
* **Power BI** – Building interactive dashboards
* **Gamma** – Creating a structured presentation/report

---

##  Project Workflow

1. **Data Loading**

   * Imported dataset into Python using Pandas

2. **Data Cleaning**

   * Handled missing values
   * Standardized column names
   * Ensured data consistency

3. **Exploratory Data Analysis (EDA)**

   * Analyzed distributions and trends
   * Identified patterns in customer behavior
   * Performed feature engineering (e.g., age groups, frequency)

4. **SQL Analysis (PostgreSQL)**

   * Executed queries to extract insights such as:

     * Revenue by customer segments
     * Top-performing products
     * Discount impact on purchases
     * Subscription vs non-subscription behavior

5. **Dashboard Creation**

   * Built an interactive Power BI dashboard to visualize key metrics

6. **Reporting & Presentation**

   * Documented findings in a report
   * Created a presentation using Gamma

---

##  Dashboard

The Power BI dashboard includes:

* Revenue analysis by category and demographics
* Customer segmentation
* Purchase trends and frequency
* Subscription insights

---

##  Results & Insights

* Identified key customer segments contributing to revenue
* Determined top-performing product categories
* Analyzed the impact of discounts on purchasing behavior
* Compared spending patterns between different customer groups

These insights can support data-driven business decisions and marketing strategies.

##  How to Run

1. Clone the repository

3. Load the cleaned data into PostgreSQL
4. Execute SQL queries for analysis
